# 🇺🇦 Волонтерська платформа для зборів

Ласкаво просимо до репозиторію нашого навчального проєкту. Ми створюємо безпечне середовище для прозорих донатів.

## 📁 Документація проєкту

Вся проєктна документація ведеться у форматі Markdown для зручності контролю версій:

1. 📜 [Хартія проєкту (Project Charter)](./docs/Project_Charter.md) — цілі та критерії успіху.
2. 🎯 [Опис обсягу (Scope Statement)](./docs/Scope_Statement.md) — що входить у розробку, а що ні.
3. 👥 [Реєстр стейкголдерів](./docs/Stakeholder_Register.md) — хто зацікавлений у проєкті.
4. 🏗️ [RACI-матриця](./docs/RACI_Matrix.md) — розподіл ролей у команді.
5. ⚠️ [План управління ризиками](./docs/Risk_Management_Plan.md) — як ми захищаємо гроші донорів.

## 🛠 Технологічний стек

- **Frontend:** HTML, CSS (Figma layout), JavaScript.
- **Backend:** Node.js / Python (в процесі).
- **База даних:** SQL-дампи знаходяться в `/backend/database/`.
