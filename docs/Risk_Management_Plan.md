# План управління ризиками

\*\*Проєкт:** Веб-платформа для благодійних / волонтерських зборів [cite: 93]
\*\*Мета:** Визначення підходу до виявлення, оцінки та реагування на ризики, пов'язані з фінансовими транзакціями та персональними даними[cite: 95, 96].

## 1. Методологія оцінки

\Кожен ризик оцінюється за формулою: **Рейтинг = Ймовірність (P) × Вплив (I)**[cite: 100].

| Рівень ризику    | Оцінка (P×I) | Дія                                    |
| :--------------- | :----------: | :------------------------------------- |
| 🔴 **Критичний** |    12-25     | \Негайне реагування [cite: 101]        |
| 🟠 **Високий**   |     8-11     | \Планові заходи реагування [cite: 101] |
| 🟡 **Середній**  |     5-7      | \Моніторинг та мітігація [cite: 101]   |
| 🟢 **Низький**   |     1-4      | \Прийняти / Спостерігати [cite: 101]   |
