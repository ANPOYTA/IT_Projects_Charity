# Scope Statement (Опис обсягу проєкту)

## 1. Цілі проєкту

- Система верифікації волонтерів для підвищення рівня довіри.
- Централізований збір коштів через банківські API.
- Прозора система звітності за кожною транзакцією.

## 2. In-Scope (Що входить в обсяг)

- **Frontend:** Сторінки зборів, кабінет волонтера, UI/UX інтерфейс для донатів.
- **Backend:** Серверна логіка, архітектура БД, обробка транзакцій.
- **Безпека:** Тестування вразливостей, захист від DDoS, платіжні шлюзи.
- **Функціонал:** Реєстрація, картки зборів, прогрес-бари накопичення коштів.

## 3. Out-of-Scope (Що не входить в обсяг)

- Логістичні послуги (закупівля/доставка товарів).
- Юридична підтримка волонтерів поза межами платформи.
- Мобільний застосунок (розробляється лише вебверсія).
- Пряма фінансова підтримка (платформа є лише посередником).

## 4. Критерії успіху

- Успішні стрес-тести.
- Стабільна робота інтеграції з банком без втрати даних.
- Позитивні результати тестування UI на зрозумілість.
