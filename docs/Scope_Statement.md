# Scope Statement (Опис обсягу проєкту)

## 1. Цілі проєкту

- Система верифікації волонтерів для підвищення рівня довіри[cite: 73].
- Централізований збір коштів через банківські API[cite: 74].
- Прозора система звітності за кожною транзакцією[cite: 75].

## 2. In-Scope (Що входить в обсяг)

- **Frontend:** Сторінки зборів, кабінет волонтера, UI/UX інтерфейс для донатів[cite: 77].
- **Backend:** Серверна логіка, архітектура БД, обробка транзакцій[cite: 78].
- **Безпека:** Тестування вразливостей, захист від DDoS, платіжні шлюзи[cite: 79].
- **Функціонал:** Реєстрація, картки зборів, прогрес-бари накопичення коштів[cite: 80].

## 3. Out-of-Scope (Що не входить в обсяг)

- Логістичні послуги (закупівля/доставка товарів)[cite: 82].
- Юридична підтримка волонтерів поза межами платформи[cite: 83].
- Мобільний застосунок (розробляється лише вебверсія)[cite: 84].
- Пряма фінансова підтримка (платформа є лише посередником)[cite: 85].

## 4. Критерії успіху

- Успішні стрес-тести безпеки[cite: 87].
- Стабільна робота інтеграції з банком без втрати даних[cite: 88].
- Позитивні результати тестування UI на зрозумілість[cite: 89].
