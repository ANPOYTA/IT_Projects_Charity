# Project Charter (Хартія проєкту)

**Назва проєкту:** Веб-платформа для благодійних/волонтерських зборів
**Керівник проєкту (РМ):** Анна Пойта

## 1. Цілі проєкту

Розробка та впровадження веб-платформи для вирішення проблем у сфері благодійності в Україні:

- **Централізація зборів:** Єдиний зручний простір для донорів.
- **Підвищення довіри:** Прозора система верифікації волонтерів та фондів.
- **Забезпечення прозорості:** Функціонал для надійної звітності щодо витрат.
- **Спрощення донату:** Зручний UI/UX та інтегровані платіжні віджети.

## 2. Критерії успіху

- Стабільна робота клієнтської (FE) та серверної (BE) частин.
- Успішна інтеграція з API банків (наприклад, підключення "банок").
- Проходження тестування безпеки (відсутність критичних вразливостей).
- Стійкість системи до DDoS-атак.
- Регулярне оновлення проєктної документації (WBS, Гант, Risk Log).

## 3. Високорівневі ризики

- **Ризики безпеки (Критичні):** Кібератаки, злам системи або пошук вразливостей.
- **Технічні ризики:** Баги, що призводять до втрати донатів або помилок у статистиці.
- **Ризики інтеграції:** Затримки при підключенні платіжних шлюзів через API.
- **Організаційні ризики:** Відхилення від таймінгу через складність логіки бази даних чи інтерфейсу.

## 4. Ключові зацікавлені сторони

- **Донори:** Користувачі, що здійснюють внески.
- **Волонтери / Благодійні фонди:** Створюють збори та надають звітність.
- **Банки та платіжні системи:** Партнери для забезпечення транзакцій.
- **Проєктна команда:** Анна Пойта, Маргарита Панченко, Анна Міллер.
